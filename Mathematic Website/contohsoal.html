Bangun Ruang

Contoh Soal dan Pembahasan

  1. KUBUS
    2.

    Suatu kubus mempunyai panjang rusuk 6 cm. Rusuk itu kemudian akan diperpanjang sebesar k kali panjang rusuk semula, sehingga volumenya berubah menjadi 1.728 cm3. Hitunglah nilai k dari panjang rusuk tersebut!

    Jawab :
    s kubus semula = 6 cm
    V kubus akhir = s x s x s = s3
    s baru = akar pangkat tiga dari 1.728 = 12
    Nilai k = 12 cm / 6 cm = 2 sehingga nilai k nya 2 kali

  2. BALOK
    3.

    Rusuk-rusuk balok bertemu pada suatu balok sebuah pojok balok berbanding 4:4:1 apabila volume balok 432 liter, luas permukaan balok yaitu

    Jawab :
    Total perbandingan dari volume = 4 x 4 x 1 = 16
    R1 = 4/16 x 432 = 108 dm
    R2 = 4/16 x 432 = 108 dm
    R3 = 1/16 x 432 = 27 dm
    R1 : R2 : R3 = 108 : 108 : 27 = 12 : 12 : 3
    Luas Permukaan
    = 2 x Luas alas + (Keliling alas x tinggi)
    = 2 x (12 x 12) + (4 x 12 x 3) (Sebab alas berbentuk persegi)
    = 288 + 144
    = 432 dm2

  3. PRISMA
    4.

    Alas dari suatu prisma berbentuk segitiga siku-siku dengan memiliki panjang sisi miring 35 cm serta panjang salah satu sisi siku-sikunya 21 cm. Jika tinggi prisma 20 cm, maka luas sisi prismanya yaitu …

    Jawab :
    Sisi tegak = A
    A2 = C2 – B2 = 352 – 212 = 1225 – 441 = 784
    A = 28 cm
    Luas sisi Prisma = 2 x Luas alas + Keliling alas x tinggi
    = 2 x (1/2 x A x B) + (A + B + C) x tinggi
    = 2 x (1/2 x 21 x 28) + (28 + 21 + 35) x 20
    = 588 + (84 x 20) = 2268 cm2

  4. LIMAS
    5.

    Diketahui sebuah limas segiempat mempunyai panjang 20 cm serta lebar 15 cm. Tinggi segitiga selimut diketahui sepanjang 10 cm. Hitunglah luas permukaan limas!

    Jawab :
    Luas Permukaan = ( p x l ) + (2 x 1/2 x p x t.selimut) + (2 x 1/2 x l x t.selimut)
    = ( 20 x 15 ) + (2 x 1/2 x 20 x 10) + ( 2 x 1/2 x 15 x 10)
    = 300 + 200 +150 = 650 cm2

  5. KERUCUT
    6.

    Tentukan volume kerucut terpancung jika diameter alasnya 10 dm, diameter sisi atas 4 dm, dan tinggi 4 dm! Jari-jari alas = 5 dm, Jari-jari atas = 2dm

    Jawab :
    Volume kerucut terpancung = phi×t ((jari alas)2 + (jari alas × jari atas) + (jari atas)2)
    = 3,14×4((5×5) + (5×2) + (2×2))
    = 12,56 (25 + 10 + 4)
    = 12,56 × 39 = 489,84 dm3

  6. BOLA
    7.

    Sebuah balon udara berwujud bola serta terbuat dari bahan elastis. Hitunglah berapa luas bahan yang dibutuhkan untuk membuat balon udara tersebut apabila diameternya 28 m dengan π=22/7!

    Jawab :
    L = 4πr2 = 4×22/7×14×14 = 2464 m2

  7. TABUNG
    8.

    Panjang jari-jari alas dari suatu tabung yaitu = 10,5 cm serta tingginya = 20 cm. Untuk π = 22/7, hitunglah: luas selimut tabung, luas tabung tanpa tutup, dan luas tabung seluruhnya

    Jawab :
    r = 10.5 cm
    t = 20 cm
    phi = 22/7
    Luas selimut tabung = 2πrt
    =2 × 22/7 × 10,5 × 20 = 1320 cm2
    Luas selimut tanpa tutup = πr2 + 2πrt
    = (22/7×10,5×10,5)+(2×π×10,5×20)
    = 346,5 + 1.320 = 1666.5 cm2
    Luas tabung seluruhnya = 2πr(r+t)
    = 2×22/7×10,5×(10,5+20) = 2013 cm2
